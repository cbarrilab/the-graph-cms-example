## Local development

### Graph local node
- https://github.com/graphprotocol/graph-node
```shell
git clone https://github.com/graphprotocol/graph-node
cd graph-node
cd docker
docker-compose up
```
Back in this repository you can run:
```shell
yarn create-local
yarn remove-local
yarn deploy-local
```
